# Energy Analysis Frontend

Frontend aplikacji do analizy i prognozowania zużycia energii zbudowany w React.

## Instalacja

```bash
npm install
```

## Uruchomienie

```bash
npm start
```

Aplikacja będzie dostępna pod adresem `http://localhost:3000`

## Funkcjonalności

- **Dashboard**: Przegląd aktualnych danych i statystyk
- **Analiza**: Panel do uruchamiania analizy zużycia energii z wykresami
- **Prognozowanie**: Panel do generowania prognoz z wizualizacją

## Technologie

- React 18
- Chart.js / react-chartjs-2
- Axios
- date-fns
