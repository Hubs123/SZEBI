## Instrukcja

GUI jest w index.html, script.js, styles.css w resources/static wystarczy przelozyc do modulu GUI

Baza danych jest h2 database, klasa, ktora odpowiada za komunikacje z baza danych to chatManager a UserRepository, ChatRepository to funkcje ktore wyciagaja z bazy danych rozne rzeczy itd

Najpierw trzeba odpalic IoCommunicationApplication a potem wejsc na http://localhost:8080/index.html
